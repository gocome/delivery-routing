# delivery-routing
Find the shortest total distance of routes to deliver demands. Code is written in Python.


### How to run code
1. Clone the repository *delivery-routing* from GitHub to a local machine, e.g., using the web URL: https://github.com/gocome/delivery-routing.git
2. Create a virtual environment with Python 3.8, and then install the required packages 
```buildoutcfg
pip install -r requirements.txt
```
3. Run the file *main.py* to get the output of delivery routs
```buildoutcfg
python main.py
```
